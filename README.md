## Updating

Briefings are copied from the internal `data/processed/YYYY-MM-DD/` output and published here after link-stripping (internal analysis links are removed). From the main Sam_news repo root: `python3 scripts/copy_briefings_to_public.py [YYYY-MM-DD]` or `--all`.
